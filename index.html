Сортировка бинарным деревом

Для начала визуализации нажмите "Начать".

Для отмены визуализации и генерации новой последовательности сортируемых чисел нажмите "Сброс".

Для отображения более детального описания нажмите "Описание".

Описание

Страница представляет собой визуализированную сортировку бинарным деревом. Для реализации использовались JQuery и d3.js. Дерево рисуется с помощью векторной графики на объекте SVG. Для организации последовательной анимации использованы очереди JQuery.

Перед началом сортировки страница генерирует массив из 38 случайных чисел от 0 до 99, который предстоит отсортировать и выводит его на страницу. Далее пользователь может начать визуализацию, нажав на кнопку "Начать".

После нажатия кнопки "Начать" происходит следующее:

  • Числа из массива для сортировки поочередно заносятся в дерево. На каждом листе дерева, начиная от корня, проверяется величина числа, если она больше величины листа, то число переходит в правую ветку дерева, а если величина меньше листа, то в левую ветку. Листы правой ветки относительно своего родителя помечаются синим цветом, а листы левой - зеленым. Путь, пройденный величиной по структуре дерева, отмечается красным. Величина листов и их значений связана, чем больше значение, тем больше лист.
  • Когда значения в массиве для сортировки чисел иссякают, начинается рекурсивный обход дерева, от самой левой ветки дерева до самой правой. Значение текущего листа сохраняется в отсортированном массиве, а сам лист помечается серым.

Как только завершится обход дерева, полностью отсортированный массив случайно сгенерированных чисел будет показан внизу страницы
